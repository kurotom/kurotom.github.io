Sobre mí

Hola, mi nombre es Tomás, disfruto crear herramientas y soluciones tecnológicas con el objetivo de resolver problemas.

Utilizo lenguajes de programación Python y JavaScript, base de datos (SQLite, PostgreSQL, MongoDB). Además creo herramientas de escritorio con entorno gráfico utilizado Tkinter, en el desarrollo web utilizo framework Django para backend (lado del servidor), Django Rest Framework para la creación de APIs y para el frontend (lado del cliente) uso la librería React, además de HTML, CSS y Bootstrap.

Utilizo Git para controlar mi código, mis trabajos están alojados en Github.
Además tengo conocimientos en análisis de datos con Python utilizando Numpy, Pandas, Matplotlib, and Seaborn para procesar y visualizar datos, web scraping.

Formándome contínuamente, investigar y aplicar nuevas técnicas para mejorar mis habilidades como desarrollador, aportar mis conocimientos a la creación soluciones según los requerimientos, trabajando en equipo para lograr los objetivos.
